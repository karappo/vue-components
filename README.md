## For developers

### Project setup
```
yarn setup
```

#### Compiles and hot-reloads for development
```
yarn start
# or
yarn serve:storybook
```

#### Compiles and minifies for production
```
yarn build:storybook
```

#### Lints and fixes files
```
yarn lint
```

#### Publish to npm registory
```
npm publish --access=public .
```
